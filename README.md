# GlobalVet Admin Panel

Panel administrativo del Club GlobalVet México.

## Acceso
URL: `https://teccapitalweb.github.io/globalvet-admin/`

## Admins autorizados
- globalvetmexico1@gmail.com
- teccapitalweb@gmail.com

## Funciones
- 📊 Resumen de miembros y estadísticas
- 👥 Gestión de miembros (activar/desactivar)
- 🎬 Agregar/eliminar videos del Club
- 📅 Editar feed semanal (lunes, miércoles, jueves...)
- 📡 Configurar próxima sesión en vivo

## Subir a GitHub
1. Crear repositorio: `globalvet-admin`
2. Subir este archivo `index.html`
3. Activar GitHub Pages → branch main → root
